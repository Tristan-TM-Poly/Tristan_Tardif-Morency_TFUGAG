# AIT Capability Absorber

`AITCapabilityAbsorber` maps open-source and API ecosystem capabilities into safe TFUGA integration plans.

## What it absorbs

It absorbs capabilities, not raw third-party code:

- interface patterns;
- orchestration ideas;
- adapter boundaries;
- documentation structure;
- test strategies;
- license metadata;
- improvement proposals.

## Inspired capability families

- graph/state-machine workflows;
- role/task multi-agent orchestration;
- retrieval/data connectors;
- tool protocols;
- API adapters;
- code-analysis and documentation agents;
- image/workflow node graphs;
- local-first assistants.

## OAK rule

```text
capability -> license check -> adapter boundary -> tests -> docs -> draft PR
```
