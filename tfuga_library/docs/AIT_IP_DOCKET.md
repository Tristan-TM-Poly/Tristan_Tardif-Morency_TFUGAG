# AIT IP Docket

`AITIPDocket` prepares patent/IP filing packets and official handoff checklists.

## Routes

- CIPO
- PCT
- USPTO

## Boundary

It prepares drafts, packets and checklists. It does not submit official filings, sign documents or pay fees.

## OAK invariant

```text
invention -> packet -> search -> claims skeleton -> review gates -> official handoff
```
