# AIT Quebec Research Absorber

`AITQuebecResearchAbsorber` maps public Quebec university research themes into TFUGA/OAK cards, synergies and proposals.

## Absorbed pattern

```text
university research theme -> capability card -> synergy -> adapter proposal -> tests/docs -> draft PR
```

## Initial synergy families

- AI/OAK Research Cartographer
- Quantum-Materials Genesis Loop
- Health + Aging + Precision OAK Loop
- Climate-Water-North Digital Twin
- Research-to-Prototype Transfer Engine

## Boundary

It absorbs public themes and interface patterns. It does not copy papers, private datasets or unpublished research.
